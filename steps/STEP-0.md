# Step 0

## Enunciado

Vamos a implementar un esquema de GraphQL que nos permita listar archivos y directorios dentro de un filesystem.

```
The_Simpsons/
├── Bart_Simpson.png
├── Lisa_Simpson.png
├── Maggie_Simpson.png
├── Father/
│   ├── Homer_Simpson.png
│   ├── Father/
│   │   └── Abe_Simpson.png
│   └── Mother/
│       └── Mona_Simpson.png
└── Mother/
    ├── Marge_Simpson.png
    ├── Patty_Bouvier.png
    ├── Selma_Bouvier.png
    ├── Father/
    │   └── Clancy_Bouvier.jpg
    └── Mother/
        └── Jacqueline_Bouvier.gif
```

## Setup

`npm install`
`npm run test:step-0`
